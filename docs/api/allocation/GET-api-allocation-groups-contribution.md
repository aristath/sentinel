# GET /api/allocation/groups/contribution

Get group contribution analysis.

**Description:**
Returns contribution analysis showing how each group contributes to portfolio performance and value.

**Request:**
- Method: `GET`
- Path: `/api/allocation/groups/contribution`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Group contribution analysis object

**Error Responses:**
- `500 Internal Server Error`: Service error
