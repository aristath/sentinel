# GET /api/portfolio/concentration

Get concentration metrics.

**Description:**
Returns portfolio concentration metrics including position concentration, sector concentration, and geographic concentration.

**Request:**
- Method: `GET`
- Path: `/api/portfolio/concentration`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Concentration metrics object

**Error Responses:**
- `500 Internal Server Error`: Service error
