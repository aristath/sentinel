# GET /api/adaptation/adaptive-parameters

Get adaptive parameters.

**Description:**
Returns adaptive market hypothesis parameters including sensitivity settings and adjustment factors.

**Request:**
- Method: `GET`
- Path: `/api/adaptation/adaptive-parameters`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Adaptive parameters object

**Error Responses:**
- `500 Internal Server Error`: Service error
