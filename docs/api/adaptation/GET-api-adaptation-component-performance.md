# GET /api/adaptation/component-performance

Get component performance metrics.

**Description:**
Returns performance metrics for different scoring components to understand which factors are performing well in the current regime.

**Request:**
- Method: `GET`
- Path: `/api/adaptation/component-performance`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Component performance metrics object

**Error Responses:**
- `500 Internal Server Error`: Service error
