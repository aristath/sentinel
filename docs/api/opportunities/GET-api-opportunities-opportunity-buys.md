# GET /api/opportunities/opportunity-buys

Get opportunity buy recommendations.

**Description:**
Returns high-quality buy opportunities based on value, quality, and technical factors.

**Request:**
- Method: `GET`
- Path: `/api/opportunities/opportunity-buys`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Array of buy opportunity objects

**Error Responses:**
- `500 Internal Server Error`: Service error
