# GET /api/planning/recommendations

Get current trade recommendations.

**Description:**
Returns the current trade plan/recommendations generated by the planning system. Includes recommended trades with scores and feasibility information.

**Request:**
- Method: `GET`
- Path: `/api/planning/recommendations`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Trade plan object with steps:
  ```json
  {
    "steps": [
      {
        "action": "BUY",
        "symbol": "AAPL.US",
        "quantity": 10.0,
        "score": 0.85,
        "reason": "Opportunity buy",
        "feasible": true
      }
    ],
    "portfolio_hash": "abc123",
    "generated_at": "2024-01-15T10:30:00Z"
  }
  ```

**Error Responses:**
- `500 Internal Server Error`: Service error
