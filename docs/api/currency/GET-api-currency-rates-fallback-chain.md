# GET /api/currency/rates/fallback-chain

Get exchange rate fallback chain.

**Description:**
Returns the fallback chain used for exchange rate retrieval, showing primary and fallback sources.

**Request:**
- Method: `GET`
- Path: `/api/currency/rates/fallback-chain`
- Parameters: None

**Response:**
- Status: `200 OK`
- Body: Fallback chain configuration object
