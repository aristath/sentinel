# T3 (Triple Exponential Moving Average T3)

Triple Exponential Moving Average T3 technical indicator.

## API Tier

**Free Tier Available**: Yes
**Premium Required**: No (but premium offers higher rate limits)

## Description

This API returns the T3 Moving Average values for a given equity. T3 is a triple-smoothed exponential moving average designed to reduce lag while maintaining smoothness.

## Endpoint

```
https://www.alphavantage.co/query
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `function` | string | Yes | Must be `T3` |
| `symbol` | string | Yes | The stock ticker symbol (e.g., `IBM`, `MSFT`) |
| `interval` | string | Yes | Time interval between data points. Valid values: `1min`, `5min`, `15min`, `30min`, `60min`, `daily`, `weekly`, `monthly` |
| `time_period` | integer | Yes | Number of data points used to calculate each T3 value (default: `5`) |
| `series_type` | string | Yes | Desired price type in the time series. Valid values: `close`, `open`, `high`, `low` |
| `vfactor` | float | No | Volume factor (default: `0.7`) |
| `datatype` | string | No | Output format. Valid values: `json` (default), `csv` |
| `apikey` | string | Yes | Your Alpha Vantage API key |

## Examples

### cURL

```bash
curl "https://www.alphavantage.co/query?function=T3&symbol=IBM&interval=daily&time_period=5&series_type=close&vfactor=0.7&apikey=YOUR_API_KEY"
```

## Rate Limits

- **Free Tier**: 25 requests per day
- **Premium Tier**: 75-1200 requests per minute (depending on plan)

## Notes

- T3 is a triple-smoothed EMA designed to reduce lag
- Volume factor (vfactor) controls the smoothing amount
- Standard parameters: time_period=5, vfactor=0.7
- More responsive than TEMA while maintaining smoothness
- Use `series_type=close` for most standard T3 calculations
