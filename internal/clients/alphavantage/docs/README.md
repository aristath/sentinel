## API Tier Information

### Free Tier

- **Rate Limit**: 25 requests per day
- **Available Endpoints**: Most endpoints are available on the free tier
- **Premium Endpoints**: `FX_INTRADAY`, `NEWS_SENTIMENT`, `REALTIME_OPTIONS`, `REALTIME_BULK_QUOTES`

### Premium Tier

- **Rate Limits**: 75-1200 requests per minute (depending on subscription plan)
- **No Daily Limits**: Premium plans remove daily request limits
- **Real-Time Data**: Access to real-time and 15-minute delayed US market data
- **Premium Features**: 
  - Real-time forex intraday data (`FX_INTRADAY`)
  - News sentiment analysis (`NEWS_SENTIMENT`)
  - Real-time options data (`REALTIME_OPTIONS`)
  - Real-time bulk quotes (`REALTIME_BULK_QUOTES`)

## Rate Limits Summary

| Tier | Daily Limit | Per Minute Limit | Premium Endpoints |
|------|-------------|------------------|------------------|
| Free | 25 requests | N/A | No |
